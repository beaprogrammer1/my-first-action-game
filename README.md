# 🚀 Galaxy Strike: Infinite Alien Survival

Galaxy Strike aik high-octane, 2D arcade shooter game hai jo **Phaser 3** framework use kar ke banayi gayi hai. Is game mein aap ko dunya ko alien invasion se bachana hai, apne ammo ka khayal rakhna hai, aur 90+ levels tak survive karna hai!

## 🎮 Live Demo
https://beaprogrammer1.github.io/my-first-action-game/

## ✨ Key Features

- **🛡️ Life System:** Aap ke paas total **3 Lives (❤️)** hain. Agar dushman takraye ya screen se niche nikal jaye, to aap ki life kam ho jaye gi.
- **📈 90+ Levels (Auto-Generated):** Har level par dushmano ki speed aur tadaad khud-ba-khud barhti hai. Level 1 se Level 100 tak ka safar challenging hai!
- **🕹️ Hybrid Controls:** 
  - **Desktop:** Use Arrow Keys to move and Spacebar to shoot.
  - **Mobile:** Dedicated on-screen touch buttons (Left, Right, Fire).
- **🔋 Ammo & Rewards:** Goliyaan limited hain! Dushmano ko maar kar **Gems (Rewards)** collect karein taake aap ka ammo refill ho sakay.
- **🎨 Dynamic Visuals:** Har level par background ka rang badalta hai taake game boring na lagay.
- **⚡ Pro-AI Movement:** Level 2 ke baad dushman **Zig-Zag** move karte hain, jis se nishana lagana mushkil ho jata hai.

## 🕹️ How to Play

1. **Start:** Main Menu par 'Start' button dabayein.
2. **Survival:** Dushmano ko niche nikalne se rokein (warna Life kam hogi).
3. **Ammo:** Blue text mein ammo check karte rahein. Agar ammo khatam ho jaye, to dushmanon se milne wale gems collect karein.
4. **Target:** Har level ka ek target kill score hota hai. Usay poora karte hi aap aglay level par promote ho jayenge.

## 🛠️ Technical Stack

- **Engine:** Phaser 3.60 (Arcade Physics)
- **Language:** JavaScript, HTML5, CSS3
- **Version Control:** Git & GitHub
